How I Work

My design process

My previous UI experience comes mainly from working on an indie game. In that project, I learned to build interfaces by starting with the overall mood of the game, then shaping the title screen, buttons, and in-game menus around one consistent visual theme.

Set the Mood

I usually begin with the title screen by creating or choosing a strong background that establishes the atmosphere and visual direction of the project.

Match the Typography

Once the background is in place, I look for a font that fits the style of the game and makes the title feel connected to the overall visual theme.

Build the Controls

I then add the functional elements the interface needs, such as a play button, settings gear, pause button, or inventory controls.

Unify the Theme

Finally, I refine the buttons, icons, and in-game menus so they share the same visual style as the title screen and feel like part of one coherent interface.
