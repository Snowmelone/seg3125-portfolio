Design 2 - Memory Game

Echo Trails

Echo Trails is a memory game where users watch a glowing trail of tiles and repeat the sequence from memory. The game was designed to practise spatial memory, sequence recall, attention and visual perception.

Open Echo Trails Prototype Back to Portfolio

Project Overview

Game Type

Echo Trails is a sequence-memory and spatial-memory game. The player watches a short visual sequence and then repeats it in the same order.

Levels

The game includes Starter, Focus and Challenge levels. Each level changes the grid size, sequence length and preview speed.

Configuration

The player can choose from Animals, Nature and Space themes before starting the game.

Personas

Maya Chen

Maya wants a quick and friendly memory game that she can play during short study breaks. She prefers simple choices, playful visuals and clear feedback.

Owen Brooks

Owen wants a more challenging game with measurable feedback. He likes performance stats, difficulty settings and a focused interface.

Design Choices

The final design combines a simple setup flow with a clear game dashboard. The interface uses strong contrast, large interactive tiles and consistent grouping to help users focus on the memory task.

  • Similarity: all tiles use the same shape and spacing.
  • Proximity: level choices and theme choices are grouped separately.
  • Figure-ground: the active tile glows against a dark background.
  • Continuity: the progress bar moves left to right as the user recalls the trail.
